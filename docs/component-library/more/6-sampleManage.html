样品管理

大设施生物设计小组2025/05/22

个人仓库

在这里，你可以管理个人的元件样品。可以创建和配置盒子、板子和容器。

仓库视图切换

在仓库管理界面，你可以通过下拉菜单在“板盒视图”和“容器视图”之间切换。

  • 板盒视图：显示所有的板子、盒子数据，适合快速浏览和操作具体盒子和板子。
  • 容器视图：显示所有的容器数据，包括已填充样品和未填充的样品，适合快速浏览和操作具体容器。

仓库筛选

在个人仓库界面，你可以通过筛选功能查看指定的数据类型，快速浏览各类数据。

  • 类型筛选：可以按板盒、容器等分类筛选数据，方便用户根据特定分类快速查找。
  • 表头筛选：在仓库列表的表头，点击各列的筛选图标，选择筛选条件，快速筛选出符合条件的数据。
  • 搜索框筛选：在仓库管理顶部的搜索框，输入关键词进行搜索，快速找到包含该关键词的板盒、容器。

TIP

  1. 三种搜索方式之间是and关系。即多种方式检索条件存在时，页面显示条件都符合的数据。
  2. 点击【重置】按钮，清空页面检索条件，刷新页面。

数据排序

在仓库列表，你可以根据个人需求，选择按照顺序对仓库列表进行排序以及对仓库列表的各列进行排序和显示控制。

  • 排序数据：根据个人需求，选择按照名称、创建时间、修改时间等排序方式对数据进行排序，方便整理和查看。
  • 列排序与显示：根据个人需要，你可以对仓库列表的各列进行排序，还可以自定义显示字段设置，选择哪些列需要显示或隐藏，以便更清晰地查看和管理仓库数据。

创建盒子

方式一：单个盒子创建

在仓库管理页面，点击「 + 」按钮，选择盒子，点击「 创建盒子 」选项。

  1. 盒子类型：给当前创建的盒子选择盒子类型（如9x9 盒、10x10 盒）。
  2. 盒子编码：可以使用盒子类型定义的前缀和顺序编号（如 box001、box002）自动生成条形码，也可以手动输入唯一编码。
  3. 盒子名称：输入盒子名称。
  4. 容器类型：给当前创建的盒子选择使用的容器类型。
  5. 元件类型：给当前创建的盒子选择储藏样品的元件类型。
  6. 盒子状态：设置是否启用该盒子。
  7. 描述：输入对该盒子的一些描述信息。

方式二：批量盒子创建

在仓库管理页面，点击「 + 」按钮，选择盒子，点击「 批量导入盒子 」选项。

在批量导入页面，通过选择要批量创建的盒子类型，再点击下载模板文件，按照模板填写要求完成录入后，再将填写好的模板上传系统批量创建。

TIP

  • 创建盒子前，你必须确保有可使用的盒子类型和容器类型，可先在数据模板配置。
  • 若选择的盒子类型有配置自定义字段，则根据设置进行输入即可。

  4. 板子类型：给当前创建的板子选择使用的容器类型。

创建容器

功能入口：

  1. 在仓库管理页面，点击「 + 」按钮，选择容器，点击「 创建容器 」选项。
  2. 在仓库管理页面，点击「 + 」按钮，选择容器，点击「 批量创建容器 」选项。
  3. 找到创建容器的盒子，进入到盒子详情页面，选择需要创建容器的孔位，点击「 创建容器 」按钮。

操作流程：

  1. 选择需要创建容器的盒子。
  2. 选择空的孔位。
  3. 点击「 下一步 」填写容器名称和容器编码。若使用容器类型定义的前缀和顺序编号（如 tube001、tube002）自动生成条形码,则将该字段留空。反之，若使用自定义编码，则输入唯一编码。

TIP

  • 创建板子时，系统会根据选择的容器类型自动生成容器。你可以自定义容器的名称和编码。

移动到

你可以将盒子容器从当前位置移动到另一个位置。找到需要移动的容器，点击「 移动到 」按钮，选择新的存储路径。

填充样品

  1. 进入板盒页面，通过与左侧的网格视图或右侧的容器列表进行交互来选择板盒中的容器，然后选择「 填充容器 」。你可以通过此操作选择一个或多个容器进行填充。
  2. 指定是从已有样品中选择进行制备填充，还是从元件中选择制备填充。
  3. 填写准备制备样品的浓度、体积。若是从已有样品中选择制备，请填写取样样品剩余体积以及是否进行了冻融。

样品取用

你可以将个人样品从指定位置取出，使用完成后再放回原位置。

功能入口：

  1. 在仓库列表，切换到“容器视图”，选择需要取用的样品。
  2. 进入板盒页面，从容器列表选择样品取用。

删除样品

当样品无法使用或已在实体仓库中清理时，你可以选择将样品从系统删除。

功能入口：

  1. 在仓库列表，切换到“容器视图”，选择需要删除的样品，点击「 删除样品 」。
  2. 进入板盒页面，从容器列表选择样品进行删除。

导出样品

方式一：选择板盒导出

在仓库列表，选择盒子、板子，系统将导出所选盒子、板子内的所有样品信息。

方式二：选择容器列表导出

在仓库列表，切换到“容器视图”，选择你到要导出的样品信息。

项目仓库

在这里，你可以查看和管理各项目组的仓库，并独立管理分配的仓库空间。在分配的空间内，支持灵活构建位置层级，包括位置、盒子、板子、容器等数据，方便高效地管理项目组的样品存储。（提供的功能和团队仓库完全一致）

功能入口：

  • 进入“资源管理”-“仓库”页面，点击【仓库类型】下拉框切换
